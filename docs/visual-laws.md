# BGME Visual Laws

## Для всех персонажей

- Один образный акцент важнее десяти мелких деталей.
- Силуэт важнее орнамента.
- Среда должна оставлять след на форме.
- Красота в BGME всегда пережившая, а не безупречная.
- Тишина важнее пафоса.

## Для форм и пропорций

- Избегать стандартной heroic fantasy-анатомии.
- Разрешены вытяжения, хрупкость, странные плечи, длинные шеи, неидеальные пропорции.
- Нельзя терять читаемость силуэта ради эксцентрики.

## Для материалов

- Использовать сочетания живого и пережившего:
  - мох
  - ткань
  - кость
  - пергамент
  - стеклянная роса
  - старый металл
  - камень
- Материал должен рассказывать, что с объектом сделало время.

## Для цвета

- Тёмный не значит грязный.
- Приглушённые зелёные, кость, серый камень, влажный коричневый, выцветшее золото.
- Яркий цвет использовать только как редкий смысловой акцент.

## Для лиц

- Не глянец.
- Не generic "красивая модель".
- Не пустая жёсткость.
- Лицо должно держать внутреннюю жизнь, странность и след памяти.

## Для костюма

- Костюм не должен ощущаться купленным, собранным по классу или фракции.
- Он должен ощущаться изменённым средой, ритуалом, походом, памятью.
- Не делаем бессмысленную детализацию ради богатства.

## Для существ и боссов

- Они не "монстры", а формы удержания памяти.
- Даже самое чудовищное существо должно выглядеть как результат среды и времени, а не как случайный хоррор-дизайн.

## Для интерфейса

- UI должен казаться архивным слоем мира.
- Стекло, пергамент, камень, выцветший металл и тихое свечение — да.
- Чистый абстрактный sci-fi или mobile-gacha язык — нет.

## Красные флаги

- generic fantasy heroine
- идеальная броня
- слишком резкий photorealistic рендер
- аниме по умолчанию
- модный, но пустой дизайн
- слишком много мелких украшений без одной сильной идеи
