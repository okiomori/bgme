# Последняя Ступень

> Тёмное indie-RPG о партии героев, стоящих на последнем рубеже.

---

## О проекте

**Последняя Ступень** — браузерная RPG с пошаговыми боями, системой партии и живым дневником мира. Проект делается силами одного человека: игровой дизайн, арт-направление и код — всё в одном репозитории.

Эстетика: мох-готика, золотые орнаменты, атмосфера уходящей эпохи.

## Запустить локально

```bash
# Python 3 уже установлен:
python -m http.server 4175
# или двойной клик:
run-game.bat
```

Открыть `http://127.0.0.1:4175` в браузере.
